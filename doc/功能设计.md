# NeoResourceManager
灵感来源[GreenResourceManager](https://github.com/klsdf/GreenResourcesManager)

## 资源分类
支持游戏，软件/工具，单图，多图/漫画/图集，电影，番剧/动画/电视剧，音声/ASMR，小说，网站等分类，并支持从以上⑨大分类派生自定义分类

## 功能列表
1. 资源crud
2. 添加资源后监听文件系统，文件修改目录时候自动更新*
3. 记录游戏运行时间 首次访问 最后访问等数据
4. 支持从贩售网站/bangumi等渠道获取资源信息

## 技术栈
1. [electron](https://github.com/electron/electron)
2. [drizzle orm](https://github.com/drizzle-orm/drizzle-orm)
3. [vue3](https://github.com/vuejs/vue-next)
4. [Naive UI](https://github.com/TuSimple/naive-ui)
